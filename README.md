# Samdesk Code Challenge

## Requirements

- Node.js 18+ (ES modules enabled)

## Running the Script

```sh
npm start
```

This runs:

```sh
node script.js
```

## Project Structure

```
.
├── script.js      # Main solution
├── input.js       # Challenge input as a JS export
├── package.json   # Minimal Node setup
└── README.md
```

## Notes

- Uses native ES modules (`import` / `export`).
- `input.js` must export the raw challenge input as a string.
- No external dependencies are required.
